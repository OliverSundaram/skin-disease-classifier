# Transfer Learning with PyTorch
I created this repo to show a website that I created that implements a fine-tuned ConvNeXt-Tiny model to classify 20 different types of skin diseases.
Sources:
 - [Dataset of skin diseases](https://www.kaggle.com/datasets/muhammadabdulsami/massive-skin-disease-balanced-dataset)
 - [Website](https://skin-disease-classifier-anpkmmpmr7nsigihxnkamf.streamlit.app/)
 
## Instructions

 - Open the link
 - The website may ask you to "wake it up", this is just because it has been inactive for a while
 - Press upload, and upload an image of your choice to detect a brain tumor
 - Once uploaded, scroll down to see a prediction, with the models confidence

## Warning
This is not a real diagnostic tool. Do not use this for real medical decisions. This is just a portfolio project.****
